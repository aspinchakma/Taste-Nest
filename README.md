# TasteNest Website - Without Responsive

---

This website was created purely for practice purposes and is not responsive. While building it, I focused on improving my CSS Grid skills, Flexbox layout techniques, and understanding of color combinations and visual design.

If you're learning front-end development, this is a great design to recreate and practice with. Try building it yourself to strengthen your layout, styling, and UI design skills.
